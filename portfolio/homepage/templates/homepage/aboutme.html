summary

i'm Arsham Ardeshirian, a software engineer based in Toronto. my path is non-linear — mechanical engineering degree first, then a deep dive into electronics, then a hard pivot into software. that breadth shows up in how i build: i think in systems, care about the physical edges (latency, queues, throughput), and own features end-to-end.

my day-job craft is production Python backends — Django, FastAPI, REST APIs, PostgreSQL — wrapped in Docker, deployed to AWS, with the boring-but-important parts in place: pytest, CI/CD, observability, and IaC via AWS CDK. AWS Certified Solutions Architect — Associate.

i go deep on things i want to truly understand. when i wanted to know how Redis works, i wrote a from-scratch async Redis server. when i wanted to know how HTTP works, i built an HTTP/1.1 server with zero dependencies. when i wanted to ship something cloud-native end-to-end, i built Log Sentinel — a real-time AWS log pipeline with Bedrock-powered anomaly detection and full IaC.

i write about what i build and break in the blog. always open to interesting roles, collaborations, and conversations.
